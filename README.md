# Dotfiles\

\
\

## Install\

\

````\

cd; mkdir -p ./code/dotfiles && curl -#L https://github.com/juanmonsalveh/dotfiles/tarball/master | tar -xzv --strip-components 1 --exclude={README.md,LICENSE,test.sh} -C ./code/dotfiles

cd ./code/dotfiles && ./bootstrap.sh

```\
````
